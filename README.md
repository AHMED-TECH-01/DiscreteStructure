# Discrete Structures Course Reference Library

This repository serves as a centralized, structured document archive for the Discrete Structures course. It contains complete lecture slides, textbook modules, reading assignments, and reference documents in PDF format, systematically organized to cover the entire theoretical curriculum of computer science mathematics.

## Course Content and Topic Coverage

The repository contents are partitioned by major learning units, tracking foundational logic through advanced algebraic structures.

### Unit 1: Foundations of Logic and Proofs
* Propositional Logic: Truth tables, logical equivalences, and propositional satisfiability.
* Predicate Logic: Quantifiers, nested quantifiers, and rules of inference.
* Proof Methodologies: Direct proofs, contraposition, contradiction, and mathematical induction.

### Unit 2: Combinatorics and Set Theory
* Fundamental Set Operations: Venn diagrams, Cartesian products, power sets, and subset proofs.
* Cardinality of Sets: Countable and uncountable sets, functions (one-to-one, onto, bijections).
* Counting Principles: Permutations, combinations, binomial coefficients, and the pigeonhole principle.

### Unit 3: Graph Theory and Relations
* Relations and Properties: Reflexive, symmetric, transitive relations, and equivalence relations.
* Graph Fundamentals: Directed/undirected graphs, paths, connectivity, and Euler/Hamiltonian circuits.
* Trees and Models: Binary trees, tree traversals, spanning trees, and algebraic applications.

### Unit 4: Number Theory and Algebraic Structures
* Divisibility and Prime Numbers: Modular arithmetic, Euclidean algorithm, and prime factorization.
* Algebraic Systems: Semigroups, monoids, groups, and Boolean algebra expressions.

## Access and Usage

### Prerequisites
A standard PDF viewer or web browser is required to read, search, and download the instructional materials.

### Local Download Instructions
To download a local copy of all lecture slides and course documents at once:
1. Clone the repository:
   ```bash
   git clone https://github.com
   ```
2. Navigate into the document tree:
   ```bash
   cd discrete-structures-course
   ```
